--config.txt 쓰는법--
config txt는 내가 보호하고 싶은 폴더 이름을 적습니다.
다른거 적지 말고 1번줄에 딱 폴더명만 적으세요.

폴더명 넣을때는
앞에 붙은거 빼고 폴더명만!!
같은 이름의 다른 위치 폴더를 두개 열지 마세요. 크래시가 날수도!
예시: C:/secret/mikuzzal/ 일때 mikuzzal을 보호대상으로 삼고싶다면
mikuzzal 만 입력합니다.

*반드시 config.txt를 먼저 정한뒤 소프트웨어를 구동하여야 합니다.*
